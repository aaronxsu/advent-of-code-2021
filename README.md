# Advent of Code 2021

Hi, welcome to Advent of Code 2021! You will find my solutions to the questions in TypeScript in this repo.

## Prerequisites

- [nvm](https://github.com/nvm-sh/nvm)
- Node version `v17.2.0`

## Before running to see my answers

- `nvm use`
- `npm install`

## To get started

- `npm start`
